## Introduction

hEdit is a pure front-end text editor. We allow users to upload, download, edit and other operations on files.

## Technical Stack

hEdit fully adopts pure front-end technology, using pure HTML, CSS and JavaScript.

## Third-party Libraries

hEdit does not use any third-party libraries for the time being. If it is used in the future, it will be added here.
